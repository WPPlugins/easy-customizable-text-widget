== Description ==

Display your widgets in style by using many customizable options this plugin offers. 
Use predefined or custom backgrounds, pins or even change fonts by selecting one of many google fonts from dropdown.

== Installation ==

This section describes how to install the plugin and get it working.

1. Install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Setup your widget in widgets section in WordPress Dashboard.
4. Enjoy!

== Screenshots ==

1. Create and use fully customizable CTA button in your widget.
2. Desing your content area by your design taste.
3. Use our predefined backgrounds or design and upload your own. Use some of our predefined pins and position them where you need them.
4. Specify Google Fonts you wish to use in your widget, each widget can use different google font.
